학습 대상

  • SAP AA를 처음 접하는 주니어 컨설턴트
  • IFRS 16 대응 프로젝트에 투입 예정인 컨설턴트
  • ECC 경험자 중 S/4HANA 전환 이후의 구조 변화를 확인하고 싶은 컨설턴트

학습 후 달성 역량

  • Chart of Depreciation / Depreciation Area 설계 원칙 설명 가능
  • Multi-GAAP(K-IFRS/IFRS/Tax) 구조에서 Area별 차이 발생 원인 판단 가능
  • IFRS 16 프로세스에서 AA의 역할 경계 명확히 구분 가능
  • S/4HANA 결산 흐름(AFAB → Universal Journal → Asset Explorer) 설명 가능

SAP 기준

  • S/4HANA 기준 (ECC 차이 부분 별도 표시)
  • New Asset Accounting(AA-FI 통합 원장) 구조 기반
  • SAP 표준 기능 범위 — 커스터마이징 내용은 별도 명시
